# A simple AI-assisted coded blog page to cater my blogging and byte content creation needs.

Started out initially as an astro page then designed and boilerplated my site via coding assistants to make my life simpler. The tool did it in React + Vite, and redid (again using coding assistants) to support native Astro blog features.

This allowed me to work on my blog content and fulfil my writing tasks. I aim to work on the architecture, deployment and maintanence of the blog.

Deployed via vercel which uses the Astro auto deploy scripts.
